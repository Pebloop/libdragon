dg_tan

This function calculate the tangente of an angle in radian.

Usage

double dg_tan(double x);
  • x : angle in radian
  • return : the tangente of angle x

Example

double dg_tan(PI/2);

Source code

double dg_tan(double x)
{
  return (dg_sin(x) / dg_cos(x));
}
