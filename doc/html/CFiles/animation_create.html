animation_create

This function is used to initialize the animation.

Usage

animation_t *animation_create(spritesheet_t *sheet, float speed);
  • sheet : spritesheet to use
  • speed : time to change frames
  • return : the initialized animation

Example

animation_t *animation = animation_create(sheet, 2);

Source code

animation_t *animation_create(spritesheet_t *sheet, float speed)
{
    animation_t *animation = malloc(sizeof(animation_t));

    if (!animation)
        return 0;
    animation->sheet = sheet;
    animation->speed = speed;
    animation->frames = 0;
    animation->size = 0;
    return animation;
}
