animator_create

This function is used to initialize an animator.

Usage

animator_t *animator_create();
  • return : the initialized animator

Example

animator_t *animator = animator_create();

Source code

animator_t *animator_create()
{
    animator_t *animator = malloc(sizeof(animator_t));

    if (!animator)
        return 0;
    animator->current = 0;
    animator->length = 0;
    animator->animations = 0;
    animator->keys = 0;
    return animator;
}
