spritesheet_free

This function is used to destroy the spritesheet.

Usage

void spritesheet_free(spritesheet_t *spritesheet);
  • spritesheet : spritesheet to destroy

Example

spritesheet_free(sheet);

Source code

void spritesheet_free(spritesheet_t *spritesheet)
{
    sfTexture_destroy(spritesheet->sheet);
    free(spritesheet);
}
