set_thickness

This function change the thickness parameter of a framebuffer, used to draw lines.

Usage

void set_thickness(framebuffer_t *fb, unsigned int thickness);
  • fb : framebuffer to update
  • thickness : new thickness

Exemple

set_thickness(fb, 10);

Source code

void set_thickness(framebuffer_t *fb, unsigned int thickness)
{
    fb->thickness = thickness;
}
