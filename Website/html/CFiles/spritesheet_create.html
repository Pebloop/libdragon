spritesheet_create

This function is used to initialize the spritesheet.

Usage

spritesheet_t *spritesheet_create(char *filepath, int x, int y);
  • filepath : path of the image
  • x : width of a tile
  • y : height of a tile
  • return : the initialized spritesheet

Example

framebuffer_t *fb = framebuffer_create(200, 300);

Source code

framebuffer_t *framebuffer_create(unsigned int width, unsigned int height)
{
    framebuffer_t *fb = malloc(sizeof(framebuffer_t));
    sfTexture *tex = sfTexture_create(width, height);
    sfSprite *sp = sfSprite_create();

    sfSprite_setTexture(sp, tex, sfFalse);
    fb->pixels = malloc(sizeof(sfUint8) * width * height * 4);
    fb->width = width;
    fb->height = height;
    fb->sp = sp;
    fb->thickness = 1;
    return fb;
}
