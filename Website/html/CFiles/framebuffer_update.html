framebuffer_update

This function is used to update the framebuffer (to display it). If not used, the changes will not appear on the window.

Usage

  1. void framebuffer_update(framebuffer_t *fb, sfRenderWindow *window);
  • fb : framebuffer to update
  • window : the window where the fb will be displayed

Exemple

  1. framebuffer_update(fb, window);

Source code

  1. void framebuffer_update(framebuffer_t *fb, sfRenderWindow *window)
  2. {
  3. sfTexture_updateFromPixels((sfTexture *) sfSprite_getTexture(fb->sp),
  4. fb->pixels, fb->width, fb->height, 0, 0);
  5. sfRenderWindow_drawSprite(window, fb->sp, NULL);
  6. }
