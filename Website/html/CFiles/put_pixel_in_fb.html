put_pixel_in_fb

This function modify a pixel of a framebuffer.

Usage

  1. int put_pixel_in_fb(framebuffer_t *fb, unsigned int x, unsigned int y, sfColor color);
  • fb : framebuffer to modify
  • x : x coordinate of the pixel
  • y : y coordinate of the pixel
  • color : color of the pixel
  • return : 0 if the change happened, 1 otherwise

Exemple

  1. put_pixel_in_fb(fb, 500, 50, sfRed);

Source code

  1. int put_pixel_in_fb(framebuffer_t *fb, unsigned int x, unsigned int y, sfColor color)
  2. {
  3. if (x > fb->width || y > fb->height)
  4. return 1;
  5. int pos = (x + y * fb->width) * 4;
  6. fb->pixels[pos] = color.r;
  7. fb->pixels[pos + 1] = color.g;
  8. fb->pixels[pos + 2] = color.b;
  9. fb->pixels[pos + 3] = color.a;
  10. return 0;
  11. }
