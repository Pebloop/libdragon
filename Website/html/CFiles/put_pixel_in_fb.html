put_pixel_in_fb

This function modify a pixel of a framebuffer.

Usage

int put_pixel_in_fb(framebuffer_t *fb, unsigned int x, unsigned int y, sfColor color);
  • fb : framebuffer to modify
  • x : x coordinate of the pixel
  • y : y coordinate of the pixel
  • color : color of the pixel
  • return : 0 if the change happened, 1 otherwise

Exemple

put_pixel_in_fb(fb, 500, 50, sfRed);

Source code

int put_pixel_in_fb(framebuffer_t *fb, unsigned int x, unsigned int y, sfColor color)
{
    if (x > fb->width || y > fb->height)
        return 1;
    int pos = (x + y * fb->width) * 4;
    fb->pixels[pos] = color.r;
    fb->pixels[pos + 1] = color.g;
    fb->pixels[pos + 2] = color.b;
    fb->pixels[pos + 3] = color.a;
    return 0;s
}
