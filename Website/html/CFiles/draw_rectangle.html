draw_rectangle

This function draw a rectangle in a framebuffer.

Usage

void draw_rectangle(framebuffer_t *fb, sfVector2u position, sfVector2u size, sfColor color);
  • fb : framebuffer to draw in
  • position : position of the rectangle (top left)
  • size : size of the rectangle
  • color : color of the rectangle

Exemple

draw_rectangle(fb, pos, size, sfBlue);

Source code

void draw_rectangle(framebuffer_t *fb, sfVector2u position,
                 sfVector2u size, sfColor color)
{
    for (int i = position.x; i < position.x + size.x; i++)
        for (int j = position.y; j < position.y + size.y; j++)
            put_pixel_in_fb(fb, i, j, color);
}
