animator_free

This function is used to destroy an animator.

Usage

void animator_free(animator_t *animator);
  • animator : animator to destroy

Example

void animator_free(animator);

Source code

void animator_free(animator_t *animator)
{
    if (animator->length > 0) {
        free(animator->animations);
        free(animator->keys);
    }
    free(animator);
}
