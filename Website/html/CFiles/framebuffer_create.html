framebuffer_create

This function is used to initialize the framebuffer.

Usage

  1. framebuffer_t *framebuffer_create(unsigned int width, unsigned int height);
  • width : width of the grid
  • height : height of the grid
  • return : the initialized framebuffer

Exemple

  1. framebuffer_t *fb = framebuffer_create(200, 300);

Source code

  1. framebuffer_t *framebuffer_create(unsigned int width, unsigned int height)
  2. {
  3. framebuffer_t *fb = malloc(sizeof(framebuffer_t));
  4. sfTexture *tex = sfTexture_create(width, height);
  5. sfSprite *sp = sfSprite_create();
  7. sfSprite_setTexture(sp, tex, sfFalse);
  8. fb->pixels = malloc(sizeof(sfUint8) * width * height * 4);
  9. fb->width = width;
  10. fb->height = height;
  11. fb->sp = sp;
  12. return fb;
  13. }
